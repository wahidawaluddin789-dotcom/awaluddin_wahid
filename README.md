## 🚀 Cara Menggunakan

### Lokal
1. Clone atau download folder ini
2. Buka `index.html` di browser
3. Edit konten sesuai kebutuhan

### Deploy ke GitHub Pages
1. Push folder ini ke repository GitHub
2. Pastikan file bernama `index.html` di root
3. Pergi ke Settings > Pages
4. Pilih "Deploy from a branch"
5. Pilih branch `main` (atau `master`)
6. Tunggu beberapa menit dan akses `username.github.io`
