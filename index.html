Setup Mudah dalam 3 Langkah

1

Buat Repository

Buat repository baru di GitHub dengan nama username.github.io

<Klik "New" di GitHub > Nama: username.github.io Pilih "Public" Klik "Create repository"

Push File Website

git clone https://github.com/username/username.github.io.git cd username.github.io echo "<h1>Hello</h1>" > index.html git add . git commit -m "Initial commit" git push origin main
